# RideFlow Operations Hub

RideFlow is a comprehensive ride management and logistics platform designed for efficient intra-city transport operations. It provides role-based dashboards for admins, dispatchers, drivers, and customers to streamline ride booking, tracking, and management.

## 🚀 Live Site
The application is live at: [https://rideflow.aliakbar.systems/](https://rideflow.aliakbar.systems/)

## ✨ Key Features

- **Role-Based Dashboards**: Tailored interfaces for Admins, Dispatchers, Drivers, and Customers.
- **Ride Management**: Create, dispatch, and track intra-city rides in real-time.
- **User & Fleet Management**: Admin tools to manage users, drivers, dispatchers, and vehicles.
- **Interactive Maps**: Real-time location tracking and route visualization using MapTiler SDK.
- **Activity Tracking**: Comprehensive logs for administrative oversight.

## 🛠 Tech Stack

- **Frontend**: React (Vite), TypeScript
- **Styling**: Tailwind CSS, Shadcn UI, Framer Motion
- **State Management**: TanStack Query (React Query)
- **Maps**: MapTiler SDK
- **Forms**: React Hook Form, Zod
- **Icons**: Lucide React

## 📦 Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   bun install
   ```
3. Run the development server:
   ```bash
   bun dev
   ```
